# Build an LLM by Sebastian Raschka
Exercises and code examples in the book "Build a Large Language Model" by Sebastian Raschka. I will be be using Python to a thorough run of the code examples and exercises in this book, followed by which I will re-implement all of my code--yeah, even the PyTorch backend--to C++ and Rust. This is the beginning of my journey into the world of language models and world models for robotics.
